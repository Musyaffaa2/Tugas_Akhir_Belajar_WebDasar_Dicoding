Pendidikan

Saya adalah mahasiswa semester 6 Teknologi Informasi di Universitas Brawijaya. Dalam perjalanan akademik saya, saya telah mempelajari berbagai aspek pengembangan perangkat lunak, dengan fokus utama pada pengembangan Front-End. Pemahaman tentang HTML, CSS, dan JavaScript menjadi dasar dalam membangun antarmuka yang menarik dan interaktif bagi pengguna.

Keahlian

Saya memiliki keahlian dalam pengembangan web menggunakan HTML, CSS, dan JavaScript. Dengan CSS, saya dapat menciptakan desain yang responsif dan estetis, sementara JavaScript membantu saya menghadirkan fitur dinamis dan interaktif. Saya juga terus mengembangkan kemampuan saya dalam menggunakan framework seperti React dan Tailwind CSS untuk meningkatkan efisiensi dalam membangun UI/UX yang modern.

Tentang Saya

Saya bercita-cita menjadi seorang Full-Stack Developer dengan fokus utama pada Front-End Development. Saya tertarik dalam menciptakan pengalaman pengguna yang optimal melalui desain yang intuitif dan fungsional. Selain itu, saya senang mengikuti perkembangan teknologi web untuk terus meningkatkan keterampilan saya dalam membangun antarmuka yang inovatif. Dunia pengembangan Front-End terus berkembang dengan hadirnya berbagai teknologi seperti React, Vue, dan Tailwind CSS. Framework ini memungkinkan pengembang untuk membangun aplikasi web yang lebih cepat, efisien, dan ramah pengguna. Dengan pendekatan komponen, pengelolaan UI menjadi lebih fleksibel dan terstruktur, membantu dalam menciptakan pengalaman pengguna yang lebih baik.
